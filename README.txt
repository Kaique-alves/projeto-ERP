Sistema ERP em Windows Form C# - README

Introdução
Bem-vindo ao projeto do Sistema ERP desenvolvido em Windows Forms utilizando C# como linguagem de programação. Este projeto está sendo criado como parte do aprendizado em sala de aula, 
abrangendo diversos conceitos fundamentais de programação, banco de dados e desenvolvimento de software.

Descrição do Projeto
O Sistema ERP é uma aplicação de Gerenciamento de Recursos Empresariais (Enterprise Resource Planning) que visa integrar e automatizar processos internos de uma empresa. 
O projeto é desenvolvido em Windows Forms para proporcionar uma interface gráfica intuitiva e fácil de usar.

Tecnologias Utilizadas
- C# (C Sharp): Linguagem de programação principal para o desenvolvimento da aplicação.
- Windows Forms: Utilizado para a criação da interface gráfica do usuário (GUI).
- Banco de Dados: MySQL WorkBench.

Funcionalidades Principais

- Cadastro de Clientes: Permite o cadastro e gerenciamento de informações dos clientes da empresa.
- Cadastro de Produtos: Permite o cadastro e gerenciamento de informações dos produtos.
- Cadastro de Cidades: Permite o cadastro e gerenciamento das cidades.
- Cadastro de Marcas: Permite o cadastro e gerenciamento das marcas.
- Cadastro de Categorias: Permite o cadastro e gerenciamento de categorias.


Pré-requisitos

- Ter MySQL instalado e configurado conforme o arquivo.

Instalação e Execução

- Clone o repositório para o seu ambiente de desenvolvimento.
- git clone https://github.com/Kaique-alves/projeto-ERP.git
- Abra o projeto na sua IDE preferida (Visual Studio, por exemplo).
- Configure o banco de dados conforme as instruções no arquivo configSQL.
- Execute a aplicação nomeada de 211469.
